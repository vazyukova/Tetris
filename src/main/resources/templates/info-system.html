Игра "Тетрис"

Информация о системе

Приложение представляет собой расширенную версию классической игры "Тетрис".

Приложение позволяет не только играть, но и:

  • Создавать новые фигуры
  • Создавать новые стаканы
  • Конфигурировать уровни сложности
  • Настраивать процесс игры

Правила игры

Тетрис — это очень популярная игра, которая появилась в 80-х. Если вы не знаете, как играть в нее, самое время научиться.

Для того, чтобы играть используйте следующую инструкцию:

  • Настройте параметры игры: выберите способ подсчета очков, и опцию показа сетки и следующей фигуры
  • Выберите уровень сложности: легкий, средний, сложный
  • Начните играть!

Правила управления фигурами:

  • Двигайте фигуру с помощью стрелок "вправо" и "влево" на клавиатуре
  • Ускоряйте фигуру с помощью стрелки "вниз"
  • Крутите фигуру с помощью стрелки "вверх"
