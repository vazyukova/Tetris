Назначение системы


Система представляет собой расширенную версию классической игры «Тетрис» с функциями администратора. Система позволяет не только играть в «Тетрис», но и:

  • Создавать новые фигуры;
  • Создавать новые стаканы;
  • Конфигурировать уровни сложности;
  • Настраивать процесс игры.

Работа с системой


Для того чтобы авторизоваться в системе, необходимо запустить приложение, после чего откроется экранная форма авторизации. Здесь нужно ввести логин и пароль, затем нажать кнопку «Войти». При успешной авторизации произойдет переход на страницу главного меню, после чего система будет ожидать действий пользователя.


Работа с системой в режиме «Игрок»


Регистрация


Для того, чтобы зарегистрироваться в системе, нужно осуществить первый вход в систему с вводом логина и пароля на странице авторизации.


Вход


Для того, чтобы войти в систему, на странице авторизациинужно ввести логин и пароль, затем нажать кнопку «Войти».


Играть в «Тетрис»


Для того, чтобы играть в «Тетрис» на странице главного меню нужно нажать на кнопку «Играть». После этого в всплывающем окне необходимо выбрать уровень сложности игры нажатием на одну из кнопок «Легкий», «Средний» или «Сложный». Когда уровень сложности выбран, откроется страница с игрой.




Манипулирование фигурами


Для того, чтобы манипулировать фигурами, используйте клавиши клавиатуры.

Правила управления фигурами:

  • Двигайте фигуру с помощью стрелок "вправо" и "влево" на клавиатуре
  • Ускоряйте фигуру с помощью стрелки "вниз"
  • Крутите фигуру с помощью стрелки "вверх"

Подсчет результата


Подсчет результата производится в том случае, если в настройка не установлено значение "не вести подсчет"

При подсчете результата по времени, прошедшее время игры отображается слева от игрового поля.

При подсчете результата по очкам, набранные очки отображаются слева от игрового поля. За каждый убраный ряд на игровом поле на «Легком» уровне сложности прибавляется 10 очков, на «Среднем» - 15 очков, на «Сложном» - 20 очков.


Конец игры


Игра закончится, когда стакан будет переполнен фигурами. После окончания игры выведется окно с сообщением о конце игры и результатом игры, если в настройка не установлено значение "не вести подсчет".

Если был произведен подсчет результатов, то можно сохранить его нажатием на кнопку "Да", после чего произойдет переход на страницу статистики. Если нажать на кнопку "Нет", то результат не будет сохранен и произойдет переход на страницу главного меню.


Настройки игры


Для того, чтобы настроить параметры игры, на странице главного меню нужно нажать на кнопку «Настройки». После этого будет открыта страница настроек игры.

Для того, чтобы во время игры в «Тетрис» отображалась сетка на игровом поле, необходимо поставить галочку в соответствующем пункте настроек.

Для того, чтобы во время игры в «Тетрис» отображалась следующая фигура, необходимо поставить галочку в соответствующем пункте настроек.

Для того, чтобы во время игры в «Тетрис» не велся подсчет результат, необходимо в выпадающем меню выбрать пункт "Не вести". Для ведения подсчета результата по времени необходимо в выпадающем меню выбрать пункт "Время". Для ведения подсчета результата по очкам необходимо в выпадающем меню выбрать пункт "Очки".


Просмотр статистики


Для того, чтобы посмотреть статистику игр, на странице главного меню нужно нажать на кнопку «Статистика». После этого будет открыта страница статистики.

Для того, чтобы посмотреть свою статистику, необходимо нажать кнопку «Моя статистика».

Для того, чтобы посмотреть статистику лучших игроков, необходимо нажать кнопку «Лучшие игроки».

Для того, чтобы вывести результаты, которые были подсчитаны по очкам, необходимо в выпадающем меню выбрать пункт "Очки". Для того, чтобы вывести результаты, которые были подсчитаны по времени, необходимо в выпадающем меню выбрать пункт "Время"



Работа с системой в режиме «Администратор»


Вход


Для того, чтобы войти в систему, на странице авторизациинужно ввести логин и пароль, затем нажать кнопку «Войти».


Работа со стаканами


Для того, чтобы начать работать со стаканами, на странице главного меню нужно нажать на кнопку «Стаканы». После этого будет открыта страница со списком стаканов.

Для того, чтобы добавить стакан, необходимо нажать кнопку «Добавить стакан». После этого откроется окно создания стакана.

Для того, чтобы удалить стакан, необходимо нажать кнопку с изображением мусорки у соответсвующего стакана.

Для того, чтобы редактировать стакан, необходимо нажать кнопку с изображением карандаша у соответсвующего стакана. После этого откроется окно создания стакана.

Создание/редактирование стакана


В окне создания/редактирования стакана можно задать/изменить значения ширины и высоты стакана.

При нажатии на кнопку «Сохранить», стакан будет сохранен(если такого стакана еще нет), после чего произойдет переход на страницу со списком стаканов.


Работа с фигурами


Для того, чтобы начать работать с фигурами, на странице главного меню нужно нажать на кнопку «Фигуры». После этого будет открыта страница со списком фигур.

Для того, чтобы добавить фигуру, необходимо нажать кнопку «Добавить фигуру». После этого откроется окно создания фигуры.

Для того, чтобы удалить фигуру, необходимо нажать кнопку с изображением мусорки у соответсвующей фигуры.

Для того, чтобы редактировать фигуру, необходимо нажать кнопку с изображением карандаша у соответсвующей фигуры. После этого откроется окно создания фигуры.

Создание/редактирование фигуры


В окне создания/редактирования фигуры можно задать/изменить форму фигуры. При нажатии на белую клетку в поле 4х4, будет изменен цвет клетки. Цветные клетки составляют фигуру. При нажатии на цветную клетку в поле 4х4, будет изменен цвет клетки на белый.

При нажатии на кнопку «Соханить», фигура будет сохранена(если она целая и такой фигуры еще нет), после чего произойдет переход на страницу со списком фигур.

Настройка уровней сложности


Для того, чтобы настроить уровень сложности, на странице главного меню нужно нажать на кнопку «Уровни сложности». После этого в всплывающем окне необходимо выбрать уровень сложности игры нажатием на одну из кнопок «Легкий», «Средний» или «Сложный». Когда уровень сложности выбран, откроется страница с настройками уровня сложности.

При нажатии на кнопку «Сохранить» на странице настройки уровня сложности система сохраняет установленные настройки и осуществляет переход на страницу главного меню.

Редактирование списка фигур


Для того, чтобы отредактировать список фигур на уровне, на странице с настройками уровня нужно нажать на кнопку «Изменить список фигур». После этого будет открыта страница со списком фигур.

Для того, чтобы добавить фигуру на уровень, нужно нажатиь на кнопку «Выбрать» у соответствующей фигуры.

Для того, чтобы удалить фигуру с уровеня, нужно нажатиь на кнопку «Удалить» у соответствующей фигуры.

При нажатии на кнопку «Сохранить» система сохраняет установленный список фигур и осуществляет переход на страницу настройки уровня сложности.

Редактирование стакана


Для того, чтобы отредактировать стакан на уровне, на странице с настройками уровня нужно нажать на кнопку «Изменить стакан». После этого будет открыта страница со списком стаканов.

Для того, чтобы выбрать стакан для уровеня, нужно нажатиь на кнопку «Выбрать» у соответствующего стакана.

При нажатии на кнопку «Сохранить» система сохраняет установленный стакан и осуществляет переход на страницу настройки уровня сложности.

Редактирование скорости падения фигур


Для того, чтобы отредактировать скорости падения фигур на уровне, на странице с настройками уровня нужно изменить положение ползунка скорости. Доступные скорости падения: 0.5, 1.0, 1.5, 2.0.
